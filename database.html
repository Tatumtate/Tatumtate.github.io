Narrative

The artifact selected for this milestone is a Customer Management System developed in C++. This application allows users to log in, view, add, update, and delete customer data.

Originally, the system stored data in memory, which meant all information was lost after the program ended. To improve this, I implemented file-based persistence, allowing data to be saved and reloaded across sessions.

I added full CRUD functionality, including the ability to create, read, update, and delete records. This enhancement reflects real-world database behavior and significantly improves system usability.

During development, I encountered challenges related to file paths and system environments. Resolving these issues strengthened my debugging and problem-solving skills.

This project demonstrates my ability to implement database concepts, improve existing systems, and manage data effectively.
